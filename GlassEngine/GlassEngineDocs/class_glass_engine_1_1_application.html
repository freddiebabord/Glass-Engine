◆ Prepare()

void GlassEngine::Application::Prepare ( )

Prepare the Vulkan renderer. This is just used to encapsulate the setup of Vulkan objects. Included in this function is the creation of command pools which contain command buffers, Creation of model, depth and shader buffers, initialisation of OpenVR (if enabled), shader descriptors, the render pass and the pipeline are also created here. The final draw commands from the models are also created here
